# Lady Adel — Image Assets

Drop the following files here before launch. The hero section and OG meta
tags expect these exact filenames.

| File                       | Purpose                              | Ideal specs                    |
| -------------------------- | ------------------------------------ | ------------------------------ |
| `lady-adel-hero.jpg`       | Hero section portrait (Section 01)   | 4:5 portrait, ≥ 1200 × 1500 px |
| `lady-adel-about.jpg`      | About section image (Section 02)     | 3:4 portrait, ≥ 900 × 1200 px  |
| `lady-adel-og.jpg`         | Social share / Open Graph preview    | 1200 × 630 px                  |

Compress to < 300 KB each (tools like Squoosh or TinyPNG work well).
If any file is missing, the hero renders a styled initials-disc fallback.
